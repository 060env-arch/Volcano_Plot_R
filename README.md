# Volcano_Plot_R
一键生成 SCI 期刊级高颜值火山图的 R 脚本
在进行组学数据分析（如多组学联用、单细胞测序）时，火山图是展示差异基因/微生物丰度最直观的方式。但很多人卡在 R 语言的参数调整上。
这个脚本提供了一个极简、可直接输出期刊发表级PDF的解决方案。
# 痛点解决
- 小白友好：自带测试数据，下载即用。
- 无需修图：直接输出高分辨率、透明背景的 PDF，可直接无损导入 Adobe Illustrator 进行最后排版。
- 颜色护眼：预设了经典的红蓝配色（可轻易修改）。

# 如何使用
1. 确保电脑已安装 R 和 RStudio。
2. 下载本仓库的所有文件放入同一个文件夹。
3. 双击打开 plot_volcano.R。
4. 全选代码并运行（Run）。
5. 在文件夹中查收你的漂亮火山图！
